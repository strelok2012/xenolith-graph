# Xen — reference assets

Static references for the Xen design system. These files are **read-only ground truth** — when in doubt how something should look, compare to these.

| File | What it shows |
|---|---|
| `nodes.svg` | All six node variants exported from Figma at 1:1 scale: default with both pins, only-inputs, three-inputs-two-outputs, without-output, rolled-up (pill, orange variant), rolled-up-without-input (pill, green variant). |
| `node-variants.png` | Same set rendered as PNG for quick visual scan. |
| `canvas-150.png` | Sample graph at 150% zoom showing edge colour-by-source-type and category-coloured headers. |
| `context-menu.png` | Context menu with "Группа нодов" / "Рефакторинг" entries and keyboard hints. |

When adding interactions or visual states that aren't documented here, capture a Figma frame, export PNG, and add it with a row in this table.
